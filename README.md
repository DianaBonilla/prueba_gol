# prueba_gol
Utilizo la libreria pygame
El programa primero crea una ventana, realiza el llamado de las dos imagenes, se otorga las posiciones respectivas de inicio, de acuerdo al balon y al arco, además tambien se la da cuantos pixeles se va a mover el balon.
Despues procedemos a realizar que solo la imagen del balon tenga movimiento con la ayudad de event.key y k_left o k_right respectivamente, para ello se debe incrementar o disminuir el movimiento en los ejes respectivos.
Problemas: No logre imprimir correctamente gol, se crea un ciclo repetitivo, y problemas al principio con el uso de la libreria.
